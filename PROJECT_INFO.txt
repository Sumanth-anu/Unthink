═══════════════════════════════════════════════════════════════════════════

1. First installation takes 5-10 minutes (downloads AI models - ~150MB)
2. Subsequent runs are much faster
3. Processing time: ~1-3 minutes for 10-minute audio

5. Maximum file size: 100MB (configurable in config.py)

🆘 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

Problem: "Module not found"
Solution: pip install -r requirements.txt

Problem: "Port already in use"
Solution: Edit .env and change PORT=5001

Problem: Slow processing
Solution: Normal for first run. Use smaller Whisper model if needed.

For more help, see GET_STARTED.md troubleshooting section.
